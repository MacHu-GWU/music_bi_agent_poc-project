- **Business Intelligence**: Data flow and analytical outputs
- **AI Intelligence**: Neural network-like connectivity

## Design Rationale

**Architectural Clarity**: Immediately communicates multi-agent system
- Diamond router at center = intelligent orchestration
- Multiple agent nodes = specialized capabilities
- Connecting lines = data flow and routing

**Functional Representation**: Each element has semantic meaning
- **Router (â—†)**: AI decision-making, intent classification, delegation
- **Agent Nodes (â—)**: SQL, Knowledge, Metrics specialists
- **Pathways**: Information flow, bidirectional communication
- **Colors**: Functional differentiation (blue=data, green=knowledge, orange=metrics)

**System Architecture Visibility**: Logo IS the architecture diagram
- Shows "agent-as-tool" pattern at a glance
- Educates users about system capabilities
- Differentiates from simple chatbot/single-agent systems

**Technical Credibility**: Appeals to developer/technical audience
- Neural network aesthetic
- Distributed systems visual language
- Microservices architecture metaphor

**Scalability**: Works across all sizes
- Full version: Complete story (128px+)
- Icon version: Essential structure (32-64px)
- Minimal version: Recognizable core (16px)

**Memorability**: Unique among AI/BI tools
- Not just another "brain" or "graph" icon
- Distinctive hub-and-spoke topology
- Color-coding creates mental model

## Usage Guidelines

### Recommended
- **Backgrounds**: Solid white (#FFFFFF) or very light gray (#F8FAFC)
- **Spacing**: Minimum 15% padding on all sides
- **Alignment**: Centered or left-aligned with text
- **Versions by Size**:
  - < 32px: Minimal version
  - 32-128px: Icon version
  - > 128px: Full version with labels

### Color Usage
- **Full Color**: Use when background supports it (white/light)
- **Monochrome**: Professional documents, print, accessibility
- **Dark Mode**: White/light gray version on dark backgrounds (#1E293B or darker)

### Animation (Digital)
- **Loading State**: Subtle pulse on router diamond
- **Active State**: Glow effect on active agent node
- **Data Flow**: Animated dots along connection pathways

### Avoid
- Stretching or distorting aspect ratio
- Reducing padding below 10%
- Using on busy/patterned backgrounds
- Rotating except 90Â° increments
- Drop shadows or 3D effects (flat design only)
- Changing individual agent colors (consistency)
